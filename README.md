# Portfolio v2

Personal portfolio website built with React, Tailwind CSS, and Framer Motion.

## Tech Stack

- React (Create React App)
- Tailwind CSS
- Framer Motion

## Scripts

- `npm install`: install dependencies
- `npm start`: run locally at `http://localhost:3000`
- `npm run build`: create production build in `build/`
- `npm test`: run tests


## Build Notes

CI runs with `CI=true`, so warnings are treated as errors.
Make sure there are no ESLint warnings before deploying.
